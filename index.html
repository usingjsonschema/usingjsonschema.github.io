The Using JSON Schema project is a set of repositories containing,

  • JSON Schema capabilities expressed through examples that can be used with command line and web tools.
  • Command line tools written in both Javascript (Node.js) and Python.

The individual project repositories can be accessed through the navigation bar.

The introductory project, jsonsyntax, is a simple command line that checks the syntax of a JSON file. It exits with a return code of zero (successful) or one (failure), making it useful for use in scripts. It can also be used as a library.

The format repository contains a simple utility for creating formatted message strings using replacement parameters.
